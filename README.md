# 🚀 Task Manager Web App

A modern Task Management web application built using **ASP.NET Core MVC** and **Entity Framework Core**.

This project allows users to manage daily tasks efficiently with a clean and responsive interface.

---

## ✨ Features

- Create tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Search tasks
- Responsive modern UI
- Dark Mode support
- Clean MVC architecture

---

## 🛠 Technologies Used

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap 5
- Bootstrap Icons
- C#
